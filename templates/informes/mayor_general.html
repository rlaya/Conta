Información del Mayor General
¿Qué es el Mayor General?

El Mayor General es un registro que muestra todos los movimientos de cada cuenta contable, permitiendo ver el historial completo y el saldo actualizado de cada una.

Características:
  • Organización por cuenta: Cada cuenta tiene su propio historial
  • Saldo acumulado: Muestra el saldo después de cada movimiento
  • Filtros: Puede verse por cuenta específica o todas
  • Exportación: Se puede exportar a Excel o PDF

  • Use cuentas específicas para análisis detallado
  • Exporte a Excel para análisis avanzado
  • Compare períodos diferentes para tendencias
  • Revise cuentas con mayor movimiento
